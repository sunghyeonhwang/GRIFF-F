- `project_members`: 프로젝트 멤버십 (project_id, user_id, role)
- `project_invites`: 초대 관리 (email/링크, member/guest 타입)

- `projects`: `share_enabled`, `invite_token` 컬럼 추가
- 회원가입 시 자동 프로필 생성 trigger (`handle_new_user`)

- 기존 유저 → profiles 자동 마이그레이션

**RLS 정책 v2**:
- `projects`: `created_by` 기반 → `project_members` 기반으로 전환
- `project_members`: 같은 프로젝트 멤버만 조회, owner만 수정/삭제
- `project_invites`: 멤버만 생성/조회, owner만 비활성화
- `comments`, `project_versions`: `project_members` 기반으로 갱신

#### 2. Auth API 확장 (auth.js)

**프로필 API**:
- `fetchProfile(userId)`: 프로필 조회
- `updateDisplayName(userId, newName)`: 이름 변경 (중복 체크)
- `checkDisplayName(name)`: 실시간 중복 확인

**멤버 관리 API**:
- `fetchProjectMembers(projectId)`: 프로필 정보 포함 멤버 목록
- `getMyRole(projectId)`: 내 역할 조회
- `updateMemberRole(memberId, newRole)`: 역할 변경 (owner만)
- `removeMember(memberId)`: 멤버 제거 (owner만)

**초대 API**:
- `inviteByEmail(projectId, email)`: 이메일 초대 (이미 가입된 유저면 즉시 추가)
- `createInviteLink(projectId, inviteType)`: 링크 생성 (member/guest 타입 선택)
- `fetchInvites(projectId)`: 초대 내역 조회
- `toggleInviteActive(inviteId, isActive)`: 초대 비활성화/활성화
- `acceptInviteByToken(token)`: 초대 토큰으로 수락
- `acceptPendingInvites(email, userId)`: 로그인 시 이메일 기반 자동 수락

**공유 관리 API**:
- `toggleShareEnabled(projectId, enabled)`: 공유 on/off
- `regenerateShareToken(projectId)`: 공유 토큰 재생성

**변경**:
- `createProject()`: owner → `project_members` 자동 등록
- `signIn()`: 로그인 시 pending invite 자동 수락
